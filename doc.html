Como usar la API

El link principal de la api es: https://fathomless-tor-77077.herokuapp.com

Para acceder a Productos desde la API deberás usar el endpoint /api/v1/products

Para acceder a Categorias desde la API deberás usar el endpoint /api/v1/categories

Para acceder a Usuarios desde la API deberás usar el endpoint /api/v1/user
